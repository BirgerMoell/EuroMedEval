# Belgian datasets

## Current seed dataset

- `entrance-be-nl`: [Belgian physician entrance exam](https://huggingface.co/datasets/jjzha/belgian-entrance-exam-physician)

## Why it is included

The Belgian seed track is worth keeping because it is:

- native-language
- derived from official public entrance exam questions
- reconstructable from a public dataset card
- broad enough to support an entrance-exam benchmark track

## Current EuroMedEval status

- task: `medical-entrance-mcq`
- tier: `silver`
- status: `unofficial`
- access mode: `script-only`

## Notes

This dataset should stay in a dedicated entrance-exam leaderboard family rather than being pooled into physician licensing or specialist aggregates.
