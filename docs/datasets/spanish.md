# Spanish datasets

## Current seed dataset

- `head-qa-es`: [HEAD-QA](https://huggingface.co/datasets/dvilares/head_qa)
- `casimedicos-es`: [CasiMedicos SQuAD](https://huggingface.co/datasets/HiTZ/casimedicos-squad)

## Why it is included

HEAD-QA is one of the best current Spanish starting points because it is built from real healthcare exam questions rather than from pure translation.

## Current EuroMedEval status

- task: `medical-knowledge-mcq`
- tier: `gold`
- status: `official`
- access mode: `script-only`

## Follow-up candidates

- official MIR-style public archives with clear reuse terms

## Notes

CasiMedicos is now supported through the non-MCQ schema path and should be evaluated separately from Spanish MCQ leaderboards when comparing task families.
