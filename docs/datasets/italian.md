# Italian datasets

## Current seed dataset

- `medschool-test-it`: [Test Medicina](https://huggingface.co/datasets/room-b007/test-medicina)

## Why it is included

Test Medicina gives EuroMedEval an Italian native exam source with real national exam style questions. It is useful even though it is not a pure physician licensing benchmark.

## Current EuroMedEval status

- task: `medical-knowledge-mcq`
- tier: `silver`
- status: `unofficial`
- access mode: `script-only`

## Notes

This dataset includes broader science and knowledge content, so it should not be weighted the same way as a clinician-targeted licensing exam once richer Italian datasets are added.
