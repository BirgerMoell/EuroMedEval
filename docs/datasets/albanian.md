# Albanian datasets

## Current seed dataset

- `entrance-al-sq`: [Albanian medical entrance exams](https://huggingface.co/datasets/marjpri/albanian-medical-exams-cycles-mcq-190)

## Why it is included

The Albanian seed track is a useful native benchmark contribution because it is:

- native-language
- extracted from official QSHA digital medicine exam material
- fully reconstructable from a public dataset card
- clearly positioned as an entrance track rather than a translated benchmark

## Current EuroMedEval status

- task: `medical-entrance-mcq`
- tier: `silver`
- status: `unofficial`
- access mode: `script-only`

## Notes

This dataset should be reported separately from physician-licensing and specialist exams, but it is still valuable for benchmarking medically relevant reasoning in Albanian.
