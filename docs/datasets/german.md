# German datasets

## Current seed dataset

- `impp-sample-de`: [IMPP Medizin Beispielaufgaben](https://www.impp.de/pruefungen/medizin/beispielaufgaben.html)

## Why it is included

The German seed track is worth carrying because it is:

- native-language
- released by the official German examination institute
- directly tied to the medical state examination format
- useful as a legally cautious bridge while fuller German sources remain hard to integrate

## Current EuroMedEval status

- task: `medical-knowledge-mcq`
- tier: `silver`
- status: `official`
- access mode: `permissioned`

## Notes

The IMPP example-task page states that the published tasks are copyrighted and that automated reuse and text/data mining are not permitted without consent. EuroMedEval therefore treats this as a manual local-import track rather than an automated downloader.
