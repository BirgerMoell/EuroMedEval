# Swedish datasets

## Current direction

Swedish is the reference language for the first wave of EuroMedEval because there is already a strong starting point in the Swedish Medical LLM Benchmark ecosystem.

## Current migrated datasets

- `smdt-sv`
- `se-em-sv`
- `se-gp-sv`
- `pubmedqa-sv`

## Migration status

These datasets are now designed to be reconstructed from the original Swedish benchmark repository rather than remaining as demo placeholders.

## Policy

The first public Swedish leaderboard should report:

- native-only Swedish score
- all-dataset Swedish score
- translated-anchor score separately
