# Portugal sourcing notes

## Best current target

- `PNA` (Prova Nacional de Acesso), the Portuguese medical specialty access examination

## Why it matters

Portugal is one of the highest-value missing countries because the PNA is a real physician post-graduate selection exam, exactly the kind of source EuroMedEval should prioritize over translated benchmarks.

## Current blocker

We have not yet found a stable, public, machine-usable archive of full question papers and answer keys that can be integrated with clear provenance. The institutional pathway appears to run through Portuguese health institutions and the Ordem dos Médicos, but the public reconstruction route is still incomplete.

## What would unlock integration

- public release of past PNA question papers with answer keys
- institutional permission to reconstruct benchmark subsets
- a stable official archive page with downloadable exam files

## Recommended next move

Treat Portugal as a priority collaboration target rather than forcing in a weak substitute dataset.
